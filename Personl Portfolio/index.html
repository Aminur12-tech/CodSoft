Plant Disease Classification Using Machine Learning and Image Processing

Description: This project focuses on classifying plant diseases through a machine learning pipeline that processes images using HSV color filtering to isolate key plant features. Using the Random Forest algorithm, the model trains on extracted image features to accurately predict disease types. This approach combines color-based feature extraction and robust classification, providing a precise and efficient method for identifying various plant diseases, essential for agricultural health monitoring and intervention.

Foodio

Description: Foodio is a modern, stylish web page for a restaurant offering a variety of services including home delivery and dine-in options. The website features an easy-to-navigate layout with quick access to sections such as the menu, online ordering, reservations, and contact details. Highlighting the restaurant's most popular dishes, including Chowmeen, Gnocchi, and Ravioli, the page showcases a menu with categorized food options like Dinner, Lunch, Dessert, and Drinks. The homepage also introduces the talented chefs behind the restaurant and offers social media links for engagement. Footer sections include additional information, such as the restaurant's location and contact details.
